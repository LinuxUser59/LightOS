Light OS isn't an operating system, rathing something imitating one in python.

Unfortunately, Light OS is targeted for Linux users only. A windows-friendly version is being planned.

Current features:
        - A fake bootloader sequence to imitate one of a real linux system
        - A "clear" command
        - A "fastfetch" systeminfo command
        - An "echo" command
        - A "pwd" command

Future releases will involve a lot more.
